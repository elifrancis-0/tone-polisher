**Tone Polisher** is an interactive web application designed to elevate written communication. It serves as an AI-powered editor, instantly transforming rough drafts or casual notes into professional, grammatically precise business text.

### Key Features
* **Interactive Web Interface:** Built with **Streamlit** for a seamless, user-friendly experience.
* **AI-Driven Refinement:** Utilizes **OpenAI's GPT-4o-mini** to analyze context and apply a "Business Professor" persona for tone correction.
* **Real-Time Processing:** Delivers instant feedback and rewritten output at the click of a button.
* **Secure Deployment:** Manages API credentials securely via environment variables.

**Tech Stack:** Python 3, Streamlit, OpenAI SDK, python-dotenv.
